# 🦷 Cheer up(치아 업) Clinic

> **청소년 스마트폰 의존도와 수면의 질이 구강 건강에 미치는 영향 분석 및 AI 예측 대시보드**  
> 본 프로젝트는 **질병관리청 '제16차 청소년건강행태조사(KYRBS 2020)'**의 50,975명 대규모 원시 데이터를 활용하여, 청소년의 스마트폰 과의존 및 수면 부족이 구강 질환 위험에 미치는 복합적 영향을 분석하고 위험도를 예측하는 머신러닝 파이프라인 및 웹 서비스입니다.

---

## 📌 프로젝트 소개 (Introduction)

단순한 예측 모델 개발에 그치지 않고, 국가 승인 통계 데이터의 특성에 맞춰 **복합표본설계 가중치(W)**를 전면 적용한 정교한 전통 통계 검정과 최신 머신러닝 기법을 결합하였습니다. 최종적으로 도출된 예측 알고리즘을 활용해, 자가진단 및 개인 맞춤형 예방 솔루션을 제공하는 **통합 헬스케어 웹 서비스**를 제안합니다.

## 🛠️ 핵심 분석 및 예측 파이프라인 (Core Pipeline)

### 1. 데이터 전처리 및 정제 (`01_Data_Preprocessing.py`)
* **원본 데이터 처리**: SAS 데이터(`.sas7bdat`) 포맷을 직접 파싱하고 불필요한 관측치 정제 및 결측치 제거 수행
* **S-Scale 스마트폰 과의존 척도**: 한국지능정보사회진흥원(NIA) 표준 스마트폰 과의존 10개 설문 문항을 합산(10~40점 범위)하여 **23점 이상을 위험군**으로 분류
* **수면의 질 범주화**: 최근 7일간 피로회복 충분 여부(5점 척도)를 기반으로 **보통~부족에 해당하는 군을 '수면 부족(Yes)'**으로 재범주화
* **기타 변수 가공**: 성별, 학교급(중/고), 성적(상/중/하), 가정편편(상/중/하), GAD-7 기반 불안 수준(없음/가벼움/중간/심각), 스트레스, 절망감, 자살생각 변수 생성

### 2. 가중치 반영 통계 분석 (`02_Statistical_Analysis.py`)
* **Rao-Scott 카이제곱 검정**: 단순 무작위 추출을 전제하는 피어슨 카이제곱 대신, 실효 표본 수($n_{eff}$)와 설계효과(DEFF)를 산출해 자유도를 보정하는 Rao-Scott 1차 보정 카이제곱 검정을 적용하여 변수 간 연관성 검정
* **다중공선성(VIF) 검증**: 위계적 회귀 모델 투입 전 다중공선성 요인을 사전 점검 ($VIF < 10$으로 다중공선성 위험 없음 판단)
* **위계적 다중 로지스틱 회귀분석**: 통제변수를 순차적으로 투입하는 3단계 위계적 모델링을 통해 변수들의 영향력을 과학적으로 입증
  * **Model 1**: 인구통계학적 요인 + 정신건강 요인
  * **Model 2**: Model 1 + 스마트폰 관련 요인 (사용시간, 의존도)
  * **Model 3 (최종)**: Model 2 + 수면의 질 요인 (Deviance, AIC, BIC 등 적합도 지표가 순차적으로 대폭 향상됨을 확인)

### 3. 탐색적 데이터 분석 시각화 (`EDA.py`)
* 가중치(W)를 반영한 인구학적 통계 분포 시각화 및 주요 요인의 단변량/이변량 분포 차트 자동 생성
* 복합 가중 스피어만 순위 상관관계(Weighted Spearman Rank Correlation) 분석을 적용해 변수 간 관계를 규명하고 히트맵 생성

### 4. 머신러닝 파이프라인 및 최적화 (`train.py`)
* **데이터셋 분할**: 데이터 불균형 및 편향 방지를 위해 Stratified 6:2:2 (Train, Validation, Test) 분할 적용
* **스케일러 튜닝**: 가중치를 반영하여 직접 구현한 **`WeightedStandardScaler`**를 이용한 피처 스케일링
* **모델 선정 및 하이퍼파라미터 최적화**: 
  * Logistic Regression, Random Forest, LightGBM, CatBoost, XGBoost 모델 후보군 모델링
  * **Optuna** TPE Sampler를 활용한 초매개변수 튜닝 및 Stratified 5-Fold Cross Validation 적용
* **임계값(Threshold) 조정**: 
  * 선별/조기 진단 목적의 의료 헬스케어 도메인 특성에 맞춰 **재현율(Recall) 85% 이상** 달성을 목표로 임계값 조정
  * 검증 데이터셋에서 **Youden's Index (Sensitivity + Specificity - 1)**를 극대화하는 최적 임계값(`0.4996`)을 산출하여 최종 예측 모델로 **XGBoost** 선정

### 5. Streamlit 웹 애플리케이션 서비스 (`app.py` / `xg_app.py`)
* **S-Scale 및 수면 상태 자가진단**: 사용자가 웹 화면에서 직접 정보를 입력하면 실시간 점수 및 그룹 계산
* **실시간 AI 구강 문제 위험도 예측**: 학습된 Weighted StandardScaler 및 XGBoost 예측기를 로드하여 3개 등급(낮음/보통/높음) 및 확률 제시
* **1:1 개인 맞춤형 행동 개선 솔루션**: 진단된 스마트폰 의존도와 구강 건강 위험군의 4가지 매트릭스 조합에 부합하는 행동 가이드 및 디지털 디톡스 수칙 처방
* **지역 기반 치과 병원 연계**: 구글 지도 API를 연동하여 주변 치과 병원을 손쉽게 탐색하고 예약 요청을 수집하는 구조 구현
* **교육용 자료실**: 청소년 스마트폰 중독 치유 프로그램(가족치유캠프, 기숙캠프, 보호자 교육 등) 안내 서비스 탑재

## 📈 주요 연구 및 예측 성능 결과

### 1. 위계적 다중 로지스틱 회귀분석 결과 (Model 3)
* **스마트폰 의존도(위험군)**는 구강 질환(통증, 잇몸 출혈 등)을 겪을 확률을 **약 1.15배** 높이며 통계적 유의성을 가짐($p < 0.001$).
* **수면의 질(부족)**은 구강 질환 발생 위험도를 **약 1.28배** 증가시키는 가장 주요한 위협 요인으로 도출됨($p < 0.001$).
* 모델 단계별로 통계 지표가 개선되어 최종 모델의 설명력과 적합도가 가장 우수한 것으로 검증됨.

### 2. 머신러닝 최종 예측 성능 (Test셋 기준)
* 목표 재현율(Recall) $\approx$ 0.85 달성을 위한 XGBoost 모델 성능 지표:
  * **최적 임계값 (Youden's Index)**: `0.4996`
  * **ROC-AUC**: `0.6385`
  * **재현율 (Recall)**: `0.8510` (이상 유무 선별력 극대화)
  * **정밀도 (Precision)**: `0.4795`

## 🚀 설치 및 시작하기 (How to Run)

### 1. 필수 라이브러리 설치
해당 리포지토리를 클론하고 아래 명령어를 이용해 의존성 패키지를 설치합니다.

```bash
pip install -r requirements.txt
```

### 2. 전체 파이프라인 순차 실행

#### (1) 데이터 전처리 실행
원본 SAS 원시자료를 전처리하여 정제된 CSV 파일을 생성합니다.
*(주의: `./data/raw/kyrbs2020.sas7bdat` 파일이 준비되어 있어야 합니다)*
```bash
python 01_Data_Preprocessing.py
```

#### (2) 가중치 교차표 검정 및 통계 위계적 회귀분석 실행
```bash
python 02_Statistical_Analysis.py
```

#### (3) 탐색적 데이터 분석 시각화 실행
```bash
python EDA.py
```

#### (4) 머신러닝 최적 모델 탐색 및 학습
Optuna 하이퍼파라미터 튜닝과 Threshold 임계값 최적화를 거쳐 최종 모델을 생성합니다.
```bash
python train.py
```

#### (5) 웹 대시보드 애플리케이션 실행
```bash
streamlit run app.py
```
> 브라우저에서 `http://localhost:8501` 주소로 실시간 접속 및 진단이 가능합니다.
